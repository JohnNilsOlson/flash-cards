JavaScript

an object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

operators are symbols that are used to perform operations on operands

Variables

hold the data value and it can be changed anytime

Variable Naming Conventions

Variable names cannot contain spaces. Variable names must begin with a letter, an underscore (_) or a dollar sign ($) Variable names can only contain letters, numbers, underscores, or dollar signs. Variable names are case-sensitive. Certain words may not be used as variable names, because they have other meanings within JavaScript.

Functions

is a block of code designed to perform a particular task

Methods

actions that can be performed on objects

Arguments

is an Array-like object accessible inside functions that contains the values of the arguments passed to that function

Parameters

the names listed in the function definition.

Return

The expression whose value is to be returned.

Chaining Methods

also known as Cascading, refers to repeatedly calling one method after another on an object, in one continuous line of code. This technique abounds in jQuery and other JavaScript libraries and it is even common in some JavaScript native methods.

Strings

a sequence of one or more characters that may consist of letters, numbers, or symbols.

Booleans

is a datatype that returns either of two values i.e. true or false.

Undefined

Any object that has been declared but not initialized or defined

NaN

represents "Not-a-Number" value. This property indicates that a value is not a legal number.

Null

is a keyword in JavaScript that signifies 'no value' or nonexistence of any value.

Alert

displays an alert box with a specified message and an OK button

Comments

used to explain the code and make the program more readable for developers.

JQuery

JavaScript library that allows web developers to add extra functionality to their websites.

Attributes

All properties have a name. In addition they also have a value.
